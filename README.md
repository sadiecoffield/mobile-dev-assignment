# TalkingTiles Mobile App

This is an ios app which helps children with special needs communicate in schools. The app provides multiple tiles that contain icons and words. When pressed, the app will say the word that is on the tile, helping the child communicate their needs.

# Features
- Add new custom tiles using the camera
- Remove custom tiles
- Add separate profiles to have personalised custom tiles for each profile
- Converts the text on each tile to speech

## Running The App

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start
   ```

3. Install Expo Go from App Store
- [Expo Go](https://expo.dev/go)

4. Scan QR code in terminal to open the app in Expo Go
